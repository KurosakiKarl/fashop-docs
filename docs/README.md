### 安装

``` bash
# 下载
wget http://statics.fashop.cn/download/fashop-v1.0.0.tar.gz

# 解压
tar -zxvf fashop-v1.0.0.tar.gz fashop

# 进到fashop项目根目录
cd fashop

# 安装
composer install

# 启动项目
php fashop start
```

::: warning 注意
请确保你的PHP 版本 >= 7.2。
:::
