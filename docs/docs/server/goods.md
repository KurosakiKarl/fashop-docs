# 商品
## 商品列表

```html
GET | POST - /goods/list
```

| 参数           | 是否必填 | 说明                                       |
| ------------ | ---- | ---------------------------------------- |
| title        | 否    | 商品名称                                     |
| category_ids | 否    | 分类id 数组格式                                |
| order_type   | 否    | 1商品价格低到高 2商品价格高到低 3销量多到少 4销量少到多 5库存多到少 6库存少到多 7创建时间早到晚 8创建时间晚到早 |
| price        | 否    | 价格区间                                     |
| page         | 否    | 页数，默认为 `1`                               |
| rows         | 否    | 条数，默认为 `8`                               |

## 商品详情

```html
GET - /goods/info
```

| 参数 | 是否必填   | 说明   |
| ---- | ---------- | ------ |
| id   | 是，二选一 | 商品id |

> 当商品的image_spec_id不为空时，代表有图片规格，需要在规格展示那儿显示，当点图图片规格时，切换图片图片；
>
> 当就一条sku并且spec[0].id === 0， 不显示规格选择，默认为该条隐藏并被选中；
