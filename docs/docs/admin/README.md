# 简述
## 接口请求
接口前缀加`/admin`，如 `POST - /shop/setBaseInfo`

域名请求时是：`https://your.domain.com/admin/shop/setBaseInfo`

端口请求时是：`http://localhost:9510/admin/shop/setBaseInfo`
